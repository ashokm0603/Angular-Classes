DOM Selectors : 
---------------
    ->DOM Selectors are used to target the HTML Elements 
    ->Types : 
        1. getElementById()  : 
                -> this used to target html Elements based on ID name 
                ->getElementById will target one single element at a time 

        2. getElementsByClassName() : 
            ->this used to target html Elements based class name 
            ->getElementsByClassName will target multiple matched classes at a time 
            -> the return  of getElementsByClassName is HTMLCollection 

        3. getElementsByTagName() 
            ->this used to target html Elements based tag name 
            ->getElementsByTagName will target multiple matched tags at a time 
            -> the return  of getElementsByTagName is HTMLCollection 

        4. getElementsByName() 
            ->this used to target html Elements based name  attribute
            -> the return  of getElementsByName is NodeList 
            

        5. querySelector() : 
            ->this used to target html Elements based id, class or tag name 
            ->querySelector will target only one element at a time 
            

        6. querySelectorAll() 
            ->this used to target html Elements based id, class or tag name 
            ->querySelectorAll will target multiple elements at a time 
            -> the return  of querySelectorAll is NodeList
