CRUD Operations In MongoDB : 
-----------------------------
    1. insert : 
            -> Insert is used to insert data into a collection 
            ->Here we have two method to insert data : 
                1. insertOne()
                    syntax: 
                        db.collectionName.insertOne({})

                2. insertMany() 
                    syntax: 
                        db.collectionName.insertMany([{} ,{} ,{}])



    2. find: 
        -> find is used to fetch or read documents 
        ->Here we have two method to fetch data : 
            1. find() 
                ->find() used to retrieve all documents present collection 
                syntax: 
                    db.collectionName.find() 

            2. findOne() 
                ->findOne() used to retrieve only one document 
                syntax: 
                    db.collectionName.findOne() 

            i. Filtering Data : 
            syntax: 
                db.collectionName.find(filterCondition)

            ex 1:  db.cse_ncrt.find({name:"ravi"})
            ex 2:  db.cse_ncrt.findOne({fees:{$gt:50000}})

            ii. Projections:
                ->used to display required fields in document 
                ->to display the fields make value true/ 1 , if doesn't display fields make value false /0 
                syntax : 
                    db.collectionName.find(filterCondition , projects )

        3.Delete :
            ->Delete is to delete the documents in collection
            ->Here we have two method to delete document :
                1. deleteOne() 
                    ->Used to Delete one Matched document in collection
                    syntax: 
                        db.collectionName.deleteOne(filterCondition) 

                    Ex: db.cse_ncrt.deleteOne({name:"ramu"})

                2. deleteMany()  
                    ->Used to Delete Many Matched documents in collection
                    syntax: 
                        db.collectionName.deleteMany(filterCondition) 
                    
                    Ex: db.cse_ncrt.deleteMany({name:"ramu"}) 


            4. Update : 
            ------------
            ->Update is used insert new fields or update values to the document
            ->here we have two methods 
                1. updateOne()
                    ->updateOne() is used to modify only on document at a time
                    ->syntax: 
                        db.collectionName.updateOne({filter condition},{$set fields}) 



                2. updateMany() : 
                    ->updateMany() is used to modify all matched  documents at a time
                    ->syntax: 

                        db.collectionName.updateMany({filter condition},{$set fields})
