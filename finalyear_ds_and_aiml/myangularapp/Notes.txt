Routes In Angular : 
-------------------
    ->Routes are used to navigate from one page to another page based on the URL path 
    ->Angular by default having router library 
    ->Types of Routes 
        1. static routes/basic routes 
                Ex: http://localhost:4200/home 
        2. nested routes 
                Ex: http://localhost:4200/product/edit
                Ex: http://localhost:4200/user/edit
                Ex: http://localhost:4200/user/view 

        3. dynamic routes /based on ID 
                Ex: http://localhost:4200/product:1
                Ex: http://localhost:4200/product:2

        4. WildCard routes /Redirected routes 
               {
                path:"**" , redirect:"/errorpage"
               }




        Syntax : 
              const routes:Routes =[
                {
                    path:"pathName" ,component:Component
                },
                {
                    path:"pathName" ,component:Component
                },
                    {}....
              ]

              @NgModule({
                        imports:[RouterModule.forRoot(routes)],
                        exports:[RouterModule]
              })


              Dynamic Routes : 
              ----------------

                ->syntax : 
                        const routes:Routes =[

                            ]

                ->If we want to take ID from the url here we need to use ActivatedRoute module
                    Ex : http://localhost:4200/product/1
                    Ex : http://localhost:4200/product/2

CRUD Operations in MongoDB: 
----------------------------
        insert : 
        ------- 
            ->insert is used to create document in collection 
            ->here we have two method to insert documents : 
                    1. insertOne() 
                        ->insertOne() is used to create / insert single document at a time 
                        ->syntax: 
                            db.collectionName.insertOne({})

                            Ex: 
                                db.studentDetails.insertOne(
                                        {
                                            name:"ravi",
                                            course:"java full stack" ,
                                            fees:50000,
                                            email:"ravi@gmail.com",
                                            skills:["java" , " react" , "spring boot"]
                                        }
                                    )

                    2. insertMany() 
                        ->insertMany() is used to create / insert multiple documents at a time 
                        ->Syntax : 
                                db.collectionName.insertMany([{},{},{},{}])

                        Ex:     
                            db.studentDetails.insertMany(
                                [
                                    {
                                        name:"ramu",
                                        fees:90000,
                                        course:"python full stack"
                                    },
                                    {
                                        name:"Rani",
                                        fees:70000,
                                        course:"MERN full stack"
                                    },
                                    {
                                        name:"Raju",
                                        fees:75000,
                                        course:"MEAN full stack"
                                    },

                                ]
                            )
            
            
            2. find : 
                -> find use to retrieve / read the document in collection
                ->Here we have 2 methods : 
                    1. find() 
                        ->find() use to retrieve / read the all documents in collection
                        syntax: 
                            db.collectionName.find()

                    2. findOne() 
                        ->findOne() use to retrieve / read only one document in collection
                        syntax: 
                            db.collectionName.findOne()


                i. Filter condition in find : 
                -----------------------------
                syntax: 
                  1. db.collectionName.find({filter condition})
                  2. db.collectionName.findOne({filter condition})


                ii. projection in find : 
                ------------------------
                ->projections is used to display required field's present document
                ->To display field's pass value 1/ true 
                ->To not  display field's pass value 0/ false

                syntax: 
                  1. db.collectionName.find({filter condition}, {projection})
                  2. db.collectionName.findOne({filter condition},{projection})
