Document: 
---------
        ->Document is the collection of fields in MongoDB
        ->Data will be store in form of BSON( binary script object notation) 
        
        
        1.Insert 
        ->To Insert Data / Document 
                1. db.collectionName.insertOne({})
                2. db.collectionName.insertMany([{},{},{},{}])


SQL: 
-----
 create table Staff(id int primary key, name varchar(20) not null , role  varchar(30), salary  int, age int)




        2. Find: 
        ->find is retrieve Data from collection
                1.findOne() 
                        -> used to retrieve one Document at a time
                        Syntax : 
                                db.collectionName.findOne() 
                        i. Filter Condition : 
                        ---------------------
                                db.collectionName.find({filter Condition})
                        ii. Projection : 
                        ------------------
                        ->To display fields pass value 1 / true 
                        ->To no need display fields pass value 0/ false
                
                                db.collectionName.find({filter Condition},{projection})

        3. Delete : 
        ------------
                -> to delete matched Document we used delete methods 
                        1. db.collectionName.deleteOne({filter condition})
                                
                        2. db.collectionName.deleteMany({filter condition})

        4.Update :
                ->Update is used to update field values , or insert new fields for existing document 
                ->If we have want to  insert fields here we need to use $set operator 

                1. updateOne({filter condition}, {updated query})
                Syntax : 
                        db.collectionName.updateOne({filter condition} ,{ updated query} )

        
                In Update Query we have Update operators : 
                        1.$set 
                        2.$unset 
                                ->It us used to remove fields from document in collection 
                                ->syntax: 
                                        db.collectionName.updateMany(
                                                {filter condition},
                                                {$unset:{removing fields }}
                                                )

                                1. WAQT remove role fields in all Staff documents who is earning 90000 rs and the name "Ravi" 

                                2.WAQT remove staff id in all Staff documents who is earning more that 80000

                        3.$pop : 
                                ->$pop operator is used remove array elements
                                ->if you to remove 0th index elements pass value -1 
                                ->if you to remove last index elements pass value 1  

                                syntax : 
                                        db.collectionName.updateMany (
                                                {filter condition},
                                                {$pop:{skills:1}}
                                        )
                                
                                        db.collectionName.updateMany (
                                                {filter condition},
                                                {$pop:{skills:-1}}
                                        )
